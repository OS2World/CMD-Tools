Dear User, Lieber Benutzer!

Please refer to CPU2.html for detailed information about CPU/2, or 
UP and MP CPU Utility/2, if you prefer the long form. I'm quite sure you have 
some ideas about that product, unless you wouldn't have downloaded it ;-).

The sourcecode is available in the encrypted file SOURCE.ZIE. By using the
program PROTECT.EXE you can decrypt it into SOURCE.ZIP by entering from an OS/2
commandline:
PROTECT cpu2v110 SOURCE.ZIE

By using encryption I can ensure that everyone interested viewing the source
code must have access to an OS/2 installation, as I'll only provide a 32-bit
OS/2 version of the decryption program (my answer to "Windows-only" vendors).

The archive SOURCE.ZIP contains the following files:
Cpu.cpp ....... Source code
Cpu.hpp ....... Source code header
Cpu.h ......... Source code header
Cpu.def ....... Module definition file
CpuUs.txt ..... Message source file (English)
CpuGr.txt ..... Message source file (German)
Make.cmd ...... Batch file to build English and German versions

Please run the executables to read the conditions of CPU/2 use!

You are welcome to drop me some comments or to visit my homepage at the URL:
http://www.geocities.com/SiliconValley/Pines/7885/

Warm regards from Austria! Roman
