APM0004I: X APM/2 позволяет переводить вашу рабочую станцию в режимы APM standby и suspend 
или выключать питание компьютера,  а также автоматически включать вашу систему 
по таймеру и восстанавливать состояние системы до вызова APM. Вызовы APM могут
назначаться встроенным планировщиком. Исходный код включает в себя необходимый
набор функций C/C++ для создания ваших собственных приложений APM для OS/2.

Эта программа является бесплатной и может свободно распространятся при  условии
сохранения целостности архива. Однако не разрешается портирование программы под
низкокачественные Windows-платформы!

APM0006I: I Параметры командной строки:

APM0007I: X Синтаксис: APM Вызов [Устройство] [Расписание] [Язык] [Версия]

Где:    Вызов        ... /Verbose   показать конфигурацию APM
                     ... /Ready     восстановить состояние до вызова APM
                     ... /Standby   включить режим APM standby
                     ... /Suspend   включить режим APM suspend
                     ... /Poweroff  выключить питание по APM
                     ... /Poweroff- вызвать DosShutdown(1) и выключить питание
                     ... /Poweroff+ вызвать DosShutdown(0) и выключить питание
                     ... /Poweron   выключить компьютер через APM по расписанию
        Устройство   ... /Device BIOS|All|Display|Disk|Parallel|Serial
                                вызов APM для указанного устройства
                                (по умолчанию All)
        Расписание   ... /Schedule [гггг:мм:дд:]чч:мм 
                                назначить вызов на определенную дату/время
        Язык         ... /Language xx (загрузить сообщения NLS из APMxx.msg)
        Версия       ... /Игнорировать обнаруженную версию OS/2

Примечание:   DosShutdown(0) ... Производит полное закрытие системы и блокирует
                         файловую систему
              DosShutdown(1) ... Сбрасывает буфферы и кэш и приводит систему в
                         состояние покоя путем прекращения выполнения тредов
